# python-hasher
A simple program that hashes a text into one of the 13 available agorithms of hashlib.

Credits:
- TheFrenchTechMan#4580 for the base code
- Liker#5894 for the optimized hashing script

Available algorithms:
- sha384
- shake_256
- sha3_224
- sha3_384
- blake2s
- sha3_256
- shake_128
- sha1
- sha224
- sha256
- sha512
- sha3_512
- md5
- blake2b
